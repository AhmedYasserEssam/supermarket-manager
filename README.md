# 🛒 Supermarket Management System

A supermarket management system developed in **C++** with **Qt5 widgets** for the graphical interface. This project simulates a real-world retail backend, supporting category and inventory management, shopping cart operations, order processing, and user accounts.

> Originally created as a CS213 course project during the first semester of sophomore year at Egypt University of Informatics.

---

## 📌 Features

- 🗂️ **Category & Product Management** using AVL Trees
- 📦 **Inventory System** with Tries and real-time stock tracking
- 🛒 **Shopping Cart & Checkout** with dynamic pricing
- 📋 **Order Management** with queue-based processing
- 👤 **User Accounts** with login, purchase history & product recommendations
- 💾 **Persistent Storage** using JSON (via `nlohmann::json`)

---

## 🧰 Tech Stack

- **Language:** C++
- **GUI Framework:** Qt5
- **Data Structures:** AVL Trees, Tries, Queues, Sets, Maps, Doubly Linked Lists
- **Persistence:** JSON
